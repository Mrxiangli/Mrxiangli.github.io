Smart Glove

Designed through Fall 2017

The concept of wearable mouse has existed for a long time. However, in most cases, people realize the control of the mouse by tilting their hand, which leads to bad user experience cause the hands are fixed and you can’t move it around. For our project, we pay more attention to the user experience. Instead of using the traditional tilting technique implemented by an accelerometer, our Wearable Mouse translates hand movements using two gyroscopes to mouse movement. Users will wear the Wearable Mouse on their hand, and move the mouse around the screen by moving their hand around in space. This kind of mouse can provide an interesting and intuitive user experience, especially with computer games requiring mouse movement, since the user now can move their hands in 3D instead of a 2D.

FPV Drone Project

- Designed during summer 2017

This a a simple FPV drone project that totally designed from scratch. Instead of buying the existing flighting control form the market I decided to design my own flight control, which embedded with a STCW series microcontroller. The design process was divided into 3 individual parts. The first part includes the theoretical design, which contains the PCB logic design and layout design. When the design is done, it will be sending to the manufacturer to manufacture. The second part includes soldering each parts into the PCB boards and testing each individual blocks' functionality. Finally, writing the code to control the operation of the drone. The detailed code for the project can be found in my Github repository.

SIMD Processor

- Designed during Spring 2018

This is a group based project for Purdue ECE 337, which is an ASIC design lab. The original idea of this project is from the NVIDIA's CUDA, which is a revolutionary parallel computing architecture. The goal of this project is to use such an parallel architecture to accelerate the data manipulation. In order to realize that, our team designed the SIMD processor, which stands for single instruction, multiple data. The processor's communication with the rest of the system is based on AHB protocol. Over all the processor is consists of 8 main modules, a register file and 8 computing units

  • AHB master    
  • AHB slave    
  • FIFO    
  • Distributor    
  • IO controller    
  • CU controller
