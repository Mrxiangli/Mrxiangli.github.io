Smart Glove

The concept of wearable mouse has existed for a long time. In most cases, the control of the mouse is achieved by tilting hands, which leads to bad user experience as the hands are fixed and can’t move around. Our project took care of this problem and enhanced the overall user experience. Instead of using the traditional tilting technique which usually accomplished by an accelerometer, our wearable mouse translates hand movements to cursor movement by using two gyroscopes. Users will wear the device on their hand and the cursor will follow the movement of their hands. This kind of mouse can provide an interesting and intuitive user experience, especially with computer games that requires mouse movement as it offers 3D movement instead of 2D movement.

SIMD Processor

This is a group project for Purdue ECE 337 - an ASIC design lab. The original idea of this project is from the NVIDIA's CUDA project, which is a revolutionary parallel computing architecture. The goal of this project is to use such an parallel architecture to accelerate the data manipulation. In order to realize that, our team designed the SIMD processor, which stands for single instruction, multiple data. The processor's communication with the rest of the system is based on AHB protocol.
